# Software FJ - Customer, Services and Reservation Management System

## Description

This project was developed for the Programming course at Universidad Nacional Abierta y a Distancia (UNAD).

The system manages customers, services, and reservations for a company called Software FJ. The application was developed using Object-Oriented Programming concepts such as abstraction, inheritance, polymorphism, encapsulation, and advanced exception handling.

The system does not use databases. All information is managed using objects, lists, and log files.

---

## Features

- Customer registration
- Service management
- Reservation processing
- Exception handling
- Error logging
- Object-Oriented Programming
- Abstract classes
- Inheritance and polymorphism
